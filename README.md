Este curso apresenta os conceitos fundamentais da Programação Orientada a Objetos (POO) utilizando Python, com exemplos práticos e exercícios progressivos.

---

### **Aula 01: O que é Programação Orientada a Objetos e o que é um Objeto?**

**Arquivo:** `poo_aula01_intro.ipynb`

Introdução ao paradigma de Programação Orientada a Objetos. Esta aula aborda:
- Conceitos fundamentais de POO como paradigma e padrão de programação
- O que são objetos e suas características (atributos/dados) e comportamentos (métodos/ações)
- Diferença entre classe (modelo genérico) e objeto (instância específica)
- Criação de classes em Python usando a palavra-chave `class`
- Criação de objetos e instanciação de classes
- Introdução ao método construtor `__init__` para inicializar atributos
- Conceitos de tipo, instância e identidade de objetos

**Exemplo prático:** Classe `Pessoa` com atributos nome, idade e profissão.

---

### **Aula 02: Atributos, Métodos e Construtores**

**Arquivo:** `poo_aula02_atributos_metodos_construtores.ipynb`

Aprofundamento nos elementos fundamentais de classes em Python. Esta aula cobre:
- Definição e uso de atributos de classe
- Criação e implementação de métodos em classes
- O conceito de `self` como referência ao próprio objeto
- Método construtor `__init__` detalhado
- Parâmetros obrigatórios e opcionais no construtor
- Diferença entre atributos de instância e variáveis locais
- Passagem de parâmetros por posição e por nome
- Valores padrão em métodos construtores

**Exemplo prático:** Classe `Pizza` com atributos (sabor, preço) e métodos (vender), incluindo lógica condicional.

---

### **Aula 03: Encapsulamento e Modificadores**

**Arquivo:** `poo_aula03_encapsulamento_modificadores.ipynb`

Conceitos de encapsulamento e controle de acesso em Python. Esta aula explora:
- Encapsulamento de dados e métodos
- Modificadores de acesso (público, privado, protegido)
- Convenções de nomenclatura em Python (`_` e `__`)
- Propriedades (`@property`) e métodos getter/setter
- Proteção de dados e controle de acesso aos atributos

---

### **Aula 04: Herança e Reutilização**

**Arquivo:** `poo_aula04_heranca_reutilizacao.ipynb`

Herança como mecanismo de reutilização de código. Esta aula apresenta:
- Conceitos de herança e hierarquia de classes
- Classes base (pai/superclasse) e classes derivadas (filho/subclasse)
- Reutilização de código através de herança
- Sobrescrita de métodos (`override`)
- Método `super()` para acessar a classe pai
- Herança múltipla em Python

---

### **Aula 05: Polimorfismo e Abstração**

**Arquivo:** `poo_aula05_polimorfismo_abstracao.ipynb`

Conceitos avançados de POO: polimorfismo e abstração. Esta aula aborda:
- Polimorfismo: múltiplas formas de implementar a mesma interface
- Classes abstratas e métodos abstratos (`abc.ABC` e `@abstractmethod`)
- Interfaces e contratos entre classes
- Duck typing em Python
- Métodos mágicos (`__str__`, `__repr__`, etc.)
- Sobrecarga de operadores

---

### **Aula 06: Projeto**

**Arquivo:** `poo_aula06_projeto.ipynb`

Projeto prático integrando todos os conceitos aprendidos. Esta aula envolve:
- Desenvolvimento de um projeto completo utilizando POO
- Aplicação prática dos conceitos de classes, objetos, herança, polimorfismo e encapsulamento
- Boas práticas de programação orientada a objetos
- Estruturação de código em classes
- Resolução de problemas reais usando o paradigma orientado a objetos

---

### **Tecnologias Utilizadas**

- **Python** - Linguagem de programação
- **Jupyter Notebook** - Ambiente de desenvolvimento interativo

### **Estrutura do Curso**

O curso é estruturado em 6 aulas progressivas, onde cada aula constrói sobre os conceitos apresentados nas anteriores, proporcionando uma aprendizagem gradual e sólida dos princípios da Programação Orientada a Objetos.
